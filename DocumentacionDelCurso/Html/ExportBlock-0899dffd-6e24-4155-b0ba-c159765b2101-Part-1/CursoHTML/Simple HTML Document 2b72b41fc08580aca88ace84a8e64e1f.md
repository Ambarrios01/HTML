# Simple HTML Document

![image.png](Simple%20HTML%20Document/image.png)

- La declaración **`<!DOCTYPE html>`** define que este documento es un documento **HTML5**.
- La etiqueta **`<html>`** es la **etiqueta raíz** de una página HTML.
- La etiqueta **`<head>`** contiene **información meta** (metadatos) sobre la página HTML.
- La etiqueta **`<title>`** especifica un **título para la página HTML** (que se muestra en la barra de título del navegador o en la pestaña de la página).
- La etiqueta **`<body>`** define el **cuerpo del documento**, y es un contenedor para **todo el contenido visible**, como encabezados, párrafos, imágenes, hipervínculos, tablas, listas, etc.
- La etiqueta **`<h1>`** define un **encabezado grande**.
- La etiqueta **`<p>`** define un **párrafo**.

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```
