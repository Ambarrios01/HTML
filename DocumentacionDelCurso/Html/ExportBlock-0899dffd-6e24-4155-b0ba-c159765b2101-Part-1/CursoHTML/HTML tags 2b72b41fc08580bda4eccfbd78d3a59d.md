# HTML tags

![image.png](HTML%20tags/image.png)

¿Cuál es la diferencia entre **tag** (etiqueta) y **element** (elemento)?

¿Qué es una **self-closing tag** o **empty tag** (etiqueta autocontenida o vacía) y qué es una **nested tag** (etiqueta anidada)?

Así que vamos al punto y empecemos con **qué es una etiqueta HTML**.

Básicamente, una etiqueta HTML es **todo lo que va desde la etiqueta de apertura hasta la etiqueta de cierre**.

Vamos a entenderlo con un ejemplo.

Esta es la **etiqueta de apertura**, esta es la **etiqueta de cierre** y este es nuestro **contenido**.

Esa etiqueta puede ser cualquier etiqueta.

Puede ser una etiqueta h1 para un encabezado grande,

puede ser una etiqueta de **párrafo**.

Aquí hay una etiqueta de encabezado:

esta es la etiqueta de apertura de **h1** y esta es la etiqueta de cierre de **h1**,

y aquí está el contenido.

El contenido puede ser cualquier cosa:

puede ser el nombre de un país, el nombre de una fruta, cualquier cosa que quieras mostrar en tu página web.

Así que vamos a entender otro ejemplo.

Esta es la **etiqueta de apertura** de la etiqueta **p**, y esta es la **etiqueta de cierre** de la etiqueta **p**.

Y este es nuestro **contenido**. El contenido es *“my first paragraph”*.

Aquí puedes colocar el contenido de tu párrafo.

![image.png](HTML%20tags/image%201.png)

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
    <h1> My First Heading</h1>
    <p> My First Paragraph</p>
    
</body>
</html>
```

![image.png](HTML%20tags/image%202.png)

El párrafo tiene **etiqueta de apertura** y también **etiqueta de cierre**.

Pero aquí puedes ver que la etiqueta **`br`** es una **etiqueta autocontenida** (*self-closing tag*).

Básicamente, las **etiquetas vacías** (*empty tags*) **no tienen etiqueta de cierre**.

Aquí puedes ver que **no hay contenido** y **no hay etiqueta de cierre**.

Entonces, `br` es una **etiqueta autocontenida o etiqueta vacía**,

y las etiquetas que **sí tienen contenido** se llaman **etiquetas en pares** (*pair tags*).
