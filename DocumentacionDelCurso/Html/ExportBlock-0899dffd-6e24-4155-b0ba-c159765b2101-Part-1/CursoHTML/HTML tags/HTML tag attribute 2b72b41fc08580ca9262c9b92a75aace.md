# HTML tag attribute

![image.png](HTML%20tag%20attribute/image.png)

### Atributo de etiqueta HTML

- Todas las etiquetas HTML pueden tener **atributos**.
- Los atributos proporcionan **información adicional** sobre los elementos.
- Los atributos siempre se especifican en la **etiqueta de apertura**.
- Los atributos normalmente vienen en **pares nombre/valor**, por ejemplo: `name="value"`.

### El atributo `lang`

Siempre debes incluir el atributo **`lang`** dentro de la etiqueta `<html>` para declarar el **idioma de la página web**.

Esto sirve para ayudar a los **motores de búsqueda** y a los **navegadores**.

El siguiente ejemplo especifica el inglés como idioma:

```html
<!DOCTYPE html>
<html lang="en">
<body>
  ...
</body>
</html>

```

Puedes ver que en esta etiqueta HTML hay un código extra, que es el **atributo de idioma** (*language attribute*).

Siempre debes incluir el **atributo de idioma** dentro de la etiqueta `<html>` para declarar el **idioma de la página web**.

Esto está pensado para ayudar a los **motores de búsqueda** y a los **navegadores**.

Entonces, en este ejemplo le estoy diciendo a mi navegador cuál es el idioma de mi página web.

Iremos aprendiendo más atributos poco a poco y usaremos diferentes atributos en nuestras etiquetas HTML.

Espero que ahora tengas una idea clara sobre los atributos, así que sigue aprendiendo.
