# Heading HTML

Sobre los encabezados. Primero, necesitamos entender qué es un encabezado.

Básicamente, los encabezados en HTML son títulos o subtítulos que quieres mostrar en una página web.

Los encabezados HTML se definen con las etiquetas desde **`<h1>` hasta `<h6>`**.

`<h1>` define el encabezado más importante y más grande, mientras que `<h6>` define el menos importante y más pequeño.

Los encabezados son muy importantes.

Los motores de búsqueda usan los encabezados para indexar la estructura y el contenido de tu página web.

Los usuarios a menudo se desplazan por una página guiándose por sus encabezados.

Es muy importante usar encabezados para mostrar la estructura del documento.

Los encabezados `<h1>` deben usarse para los títulos principales, seguidos por `<h2>`, luego los menos importantes `<h3>`, y así sucesivamente.

Hay algo muy importante que recordar:

Usa los encabezados **solo para encabezados**.

No uses encabezados solo para hacer el texto más grande o en negrita.

Cada encabezado tiene un tamaño predeterminado.

Pasemos a la parte de código.

Voy a abrir mi editor de código Visual Studio Code, aquí está mi directorio de trabajo, así que voy a crear un archivo nuevo haciendo clic derecho.

Luego en “New File” (nuevo archivo) y le doy un nombre:

**headings.html**

La extensión **`.html`** es la extensión de archivo que indica que este archivo es un archivo HTML.

Hago clic en este archivo y se abre en el editor, y aquí presiono **Shift + signo de exclamación**.

Es decir, **Shift + 1**, y entonces escribe el signo de exclamación.

Ahora simplemente presiono la tecla **Tab**.

Esto crea una estructura básica de página.

Y aquí está nuestra área de escritura.

Para crear un encabezado grande, necesitamos usar la etiqueta `<h1>`.

Así que simplemente escribo `<h1>` y luego presiono **Tab**.

Esto creará la etiqueta de apertura y de cierre para `<h1>`.

Y aquí colocaré un texto. Si presiono **Ctrl + S**, se guardará el archivo y luego abriré este archivo.

Voy a abrir el archivo.

Aquí está.

> 
> 
> (Soy encabezado 1).
> 

Ahora, si escribo `<h2>` y luego presiono Tab, y aquí escribo el texto:

> I am heading 2
> 
> 
> (Soy encabezado 2).
> 

Y si creo un `<h3>` con el texto *heading 3* (encabezado 3),

presiono **Ctrl + S**, y luego actualizo la página, puedes ver el resultado.

Aquí está el primer encabezado más grande, luego un encabezado un poco más pequeño, y aquí está el tercer tipo de encabezado.

Si creo un `<h4>` para *heading 4*,

`<h5>` con el texto *I am heading 5*,

`<h6>` con el texto *I am heading 6*,

presiono **Ctrl + S**,

cuando actualizo la página, puedes ver los resultados.

Puedes ver todos los tipos de encabezados.

Este es un encabezado grande y este es un encabezado más pequeño.

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>
  <h1>I am a Heading 1</h1>
  <h2>I am a Heading 2</h2>
  <h3>I am a Heading 3</h3>
  <h4>I am a Heading 4</h4>
  <h5>I am a Heading 5</h5>
  <h1>I am a Heading 6</h1>
</body>
</html>
```
